Smart Attendence System using Sound waves

Sep 21, 2019

This project was done as a part of my academic curriculum. The flow of project is as follows:

1)Teacher walks in the classroom and open mobile app. They can signed in by their teacher id.

2)Students can signed in the same application by using their student id.

3)Teacher will press a sound emmiting button in their smart phone which will emit sound waves that can be received by students smart phones. Once the sound waves is received then the attendence of student will be marked.

The technology included in the project are: Android studio for designing mobile app, java language for coding, MySQL database for storing student records, and RSA encryption algorithm for maintaining the confidentiality of students.
